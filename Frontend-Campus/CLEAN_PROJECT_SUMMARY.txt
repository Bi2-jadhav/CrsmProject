===============================================================
HOW TO USE
===============================================================

1. DOWNLOAD
   • Click "Download ZIP" in v0 top-right corner
   • Extract the project

2. INSTALL
   • cd recruitment-platform
   • npm install

3. SETUP
   • Create .env.local file
   • Add: NEXT_PUBLIC_API_URL=http://localhost:8080

4. RUN
   • npm run dev
   • Visit http://localhost:3000

5. TEST
   • Use role selection at signup
   • Login with test accounts
   • Explore each dashboard

===============================================================
API INTEGRATION
===============================================================

All requests use Axios with automatic JWT token injection

Base URL: http://localhost:8080 (configurable)

Endpoints used:
• POST   /api/auth/signup
• POST   /api/auth/login

• POST   /api/student/apply
• GET    /api/student/applications
• POST   /api/company/jobs
• GET    /api/company/applicants
• PUT    /api/company/applicants/{id}
• GET    /api/admin/users
• GET    /api/admin/companies
• GET    /api/admin/analytics
