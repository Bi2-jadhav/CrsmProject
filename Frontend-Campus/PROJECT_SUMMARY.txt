===============================================
SETUP & INSTALLATION
===============================================

1. Install Dependencies:
   npm install

2. Configure API URL:
   Edit src/context/AuthContext.jsx (line ~6)
   Edit src/utils/api.jsx (line ~3)
   Change: http://localhost:8080 → your API URL

3. Start Development Server:
   npm run dev
   (Opens automatically at http://localhost:3000)

4. Build for Production:
   npm run build
   (Output in dist/ folder)

5. Preview Production Build:
   npm run preview
